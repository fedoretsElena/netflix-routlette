# netflix-routlette


## Running the app

### Serve
```bash
# install dependencies
yarn install
# serve app
npm run start
```
Go [http://localhost:8081/](http://localhost:8081/) in your browser

### Build
#### Default
```bash
npm run build
```
#### Development / Production
```bash
npm run build:dev
npm run build:prod
```

### Tests
```bash
npm run test
```
